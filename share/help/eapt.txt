eapt - apt wrapper.

Usage: eapt <command> ...

command:

  upgrade     : Update and upgrade.
  full-upgrade: Update and full-upgrade.
              : After that, remove unrequired packages and caches.
  install     : Update and upgrade and install packages.
  help        : Display help text.

If a command other than the above is given, it is given to 'apt'.
If a command is not given, display help text.

apt help
